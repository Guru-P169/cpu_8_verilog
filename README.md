# 8-bit CPU Design in Verilog

## Overview

This project implements a basic 8-bit CPU using Verilog HDL. The design includes essential components such as an ALU, register operations, and control logic. The CPU functionality is verified through simulation and schematic representation.

## Tools Used

* Quartus Prime (Design & Synthesis)
* EDA Playground (Simulation)
* Xilinx Tools (Verification & Testing)

## Features

* 8-bit ALU (Addition and Subtraction operations)
* Modular CPU design
* Testbench for functional verification
* Simulation waveform output
* RTL schematic visualization

## Project Files

* `cpu8.v` → Main CPU design module
* `cpu8_tb.v` → Testbench for simulation
* `cpu8_RTL_schematic.png` → RTL schematic diagram
* `cpu8_simulation.png` → Simulation waveform output
* `cpu_8bit_architecture_design.pdf` → Detailed architecture documentation

## Simulation

The CPU design is tested using a Verilog testbench. Simulation results confirm correct execution of ALU operations.

## Architecture

The CPU architecture includes:

* Arithmetic Logic Unit (ALU)
* Control Logic
* Data path operations
